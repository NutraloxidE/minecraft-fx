# Adminから実行停止をできる体裁取引のサーバー管理スクリプト
- 管理画面で、体裁取引に使用するアカウントを選択できる
- 管理者はいつでもスクリプトの実行停止を指示できる
- 一般的なアービトラージ取引スクリプトとして動作し、デフォルトで15秒に1回検査する
- 全ペアを対象に監視し、取引可能なペアごとに機械的に判定する
- アイテムペアが2つ以上ある場合にのみ実行される
- 価格差が機械的に成立している場合のみ発注する
- 最後に設定したリミットが、直近の最終約定価格から離れすぎている場合はキャンセルする
- 検知可能な範囲で板操作や不自然な変動が見られた場合は取引を行わず、その回はスリップ扱いにする
- スリップ時は無理に追撃せず、次回の定期検査まで待機する
